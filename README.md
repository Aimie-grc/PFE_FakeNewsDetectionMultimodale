# PFE_FakeNewsDetectionMultimodale
Système de détection de fake news multimodal combinant texte et image. Utilise DistilBERT, ResNet-50 et une fusion des représentations, avec enrichissement optionnel via RAG. API FastAPI et interface Streamlit permettant l’inférence et la visualisation des résultats avec éléments d’explicabilité.
